## Setup Geral

Cada exercício é um projeto Vue independente. Para executar:

```bash
# Entrar na pasta do exercício
cd counter-app
# ou
cd watchlist-app

# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev
```
